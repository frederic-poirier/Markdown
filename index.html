Simple applis de note

Une bibliothèque légère de rendu Markdown en temps réel pour le web, axée sur une manipulation fine du DOM. Ce projet offre une édition fluide de Markdown avec une prédiction instantanée des changements, tout en gardant une structure basée sur le DOM, sans besoin de re-parsing global.
